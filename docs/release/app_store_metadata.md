## Description

Aloe Wellness Coach допомагає підтримувати спокійний щоденний ритм навколо води, сну, настрою та звичок.

У застосунку ви можете:
- обрати wellness-цілі й пройти м’який вступний сценарій
- отримати 7, 14 або 21-денний персональний план
- відстежувати воду, сон, вагу, настрій і щоденні нотатки
- бачити серію днів, виконання плану та базову аналітику
- отримувати м’які підказки щодо категорій продуктів у форматі загальної wellness-підтримки
- відкривати сторінки продуктів і магазин на зовнішньому сайті

Aloe Wellness Coach створений як застосунок для wellness- і lifestyle-підтримки.

Важливо:
- застосунок не є медичним сервісом або медичним пристроєм
- він не встановлює діагноз, не лікує та не обіцяє медичних результатів
- усі рекомендації мають загальний wellness-характер
- при симптомах, медичних станах, вагітності, алергіях, питаннях щодо ліків або сильній реакції на продукт слід звернутися до кваліфікованого медичного фахівця
