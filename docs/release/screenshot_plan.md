# Screenshot Plan

## iPhone / Android Primary Set

1. Onboarding hero
   Покажіть polished hero з layered aloe gradients, преміальними cards і чіткою typographic hierarchy.
2. Goals + questionnaire
   Демонструє premium goal cards, calm selection states і wellness-safe позиціонування без медичних обіцянок.
3. Dashboard
   Water ring hero, metric cards, сьогоднішній огляд, quick actions та product recommendations.
4. Progress
   Sleep, mood, notes, elevated analytics chart і streak-oriented metrics.
5. Plans
   7/14/21-day structure, elegant plan cards, daily checklist, adherence progress.
6. Product detail
   Usage notes, caution notes, refined tag chips і polished CTA pair for in-app/browser opening.
7. Profile + legal
   Theme controls, reminders, legal hub, support actions і premium row styling.
8. Dark mode showcase
   Окремо підготуйте dashboard або progress screen у dark mode з м’яким контрастом і читабельними surfaces.

## Visual Guidance

- Використовуйте реальні seeded дані замість порожніх екранів.
- Показуйте український текст.
- Уникайте default-looking layouts, prototype placeholders і випадкових системних кольорів.
- У кожному кадрі тримайте consistent radius, shadows, spacing rhythm і aloe-inspired palette.
- Підготуйте окремі світлі та темні кадри для маркетингового пакета.
- Для onboarding і dashboard використовуйте screens із найсильнішим hero-станом, а не компромісні проміжні кадри.
- Для App Store / Google Play додайте щонайменше один кадр із legal-safe wording про wellness support.
- Не використовуйте на скріншотах wording, що натякає на діагностику, лікування, медичний ефект або гарантований результат.
