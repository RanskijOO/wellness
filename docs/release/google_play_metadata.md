# Google Play Metadata Draft

## App Name

Aloe Wellness Coach

## Short Description

Щоденні wellness-плани, вода, сон, настрій і м’яка підтримка звичок.

## Full Description

Aloe Wellness Coach допомагає підтримувати спокійний wellness-ритм без перевантаження. Оберіть цілі, отримайте персональний план і відстежуйте прогрес у зручному мобільному форматі.

Що всередині:
- вступний сценарій із wellness-цілями, базовими звичками, згодою та безпечним дисклеймером
- 7 / 14 / 21-денні програми для щоденної послідовності
- трекери води, сну, ваги, настрою та щоденних нотаток
- серія днів, виконання плану та базова аналітика
- м’які нагадування та вечірній щоденний огляд
- загальні wellness-підказки щодо категорій продуктів
- відкриття сторінок продуктів і магазину через зовнішній сайт

Позиціонування:
- Aloe Wellness Coach є застосунком для wellness- і lifestyle-підтримки
- застосунок не є медичним продуктом або медичним пристроєм
- він не призначений для діагностики, лікування чи профілактики хвороб
- усі рекомендації мають загальний wellness-характер

Додатково:
- частина функцій доступна в гостьовому режимі
- юридичні екрани доступні з екранів входу та вступного сценарію
- покупки продуктів не виконуються всередині MVP-додатка
